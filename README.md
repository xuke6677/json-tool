# 开发工具箱

基于 Electron 的桌面端开发工具集合，面向开发者的日常效率工具。纯原生 HTML/CSS/JS 实现，无框架依赖，启动快、体积小。

## 📦 下载安装

前往 [Releases](https://github.com/xuke6677/development_toolkit/releases) 页面下载最新版 `开发工具箱 Setup x.x.x.exe`，双击安装即可使用。

## 🛠️ 功能模块

### 1. JSON 格式化

全功能 JSON 编辑器，左右双栏布局，支持大数据量流畅操作。

- 格式化 / 压缩 / 校验 JSON
- 语法高亮（key、string、number、boolean、null 分色）
- 折叠 / 展开节点，支持全部折叠 / 全部展开
- 内联编辑：点击 key 或 value 直接修改，自动同步
- 转义 / 去转义处理
- 转 Java Bean：JSON 结构一键生成 Java 类代码，支持嵌套
- 左右独立搜索栏，支持关键字搜索、上下导航、计数
- 时间戳悬停识别：鼠标悬停 13 位时间戳自动显示可读时间
- 虚拟滚动：右侧输出区只渲染可视区域，DOM 节点数可控
- 可拖动分隔条调整左右面板宽度
- 底部状态栏：字符数、行数、数据大小

### 2. 时间戳转换

实时时间戳显示与双向转换工具，默认北京时间。

- 实时显示当前时间戳（毫秒/秒可切换），支持开始/停止
- 时间戳 → 日期时间：输入或粘贴时间戳自动转换
- 日期时间 → 时间戳：输入 `yyyy-MM-dd HH:mm:ss` 格式自动转换
- 快捷下拉：一键填入当前日期或当前时间
- 毫秒/秒单位切换
- 一键复制，操作反馈提示
- 附带时间戳简介、北京时间夏令时说明、JDK 夏令时问题参考

### 3. 翻译

中英文互译 + 简繁转换，输入即翻译。

- 支持英文、中文、繁体三种输入模式切换
- 中英互译基于 MyMemory 免费 API
- 简繁转换为离线本地处理，无需网络
- 输入或粘贴后自动触发翻译（500ms 防抖）
- 双结果框同时展示两种转换结果

### 4. SQL 拼接工具

批量值快速转换为 SQL 常用格式。

- 每行一个值，实时生成四种格式：
  - 单引号包裹逗号分隔：`'a', 'b', 'c'`
  - 双引号包裹逗号分隔：`"a", "b", "c"`
  - 纯逗号分隔：`a, b, c`
  - 数组格式：`[a, b, c]`
- 合并一行 / 去重 / 清空
- SQL 格式化：关键字大写、自动换行缩进
- 各结果一键复制

### 5. Cron 生成器

可视化 Cron 表达式配置工具。

- 秒、分、时、日、月、周、年 七个字段独立配置
- 支持每值、区间、周期、指定值等多种模式
- 周字段支持第 N 个星期几（#）、最后一个星期几（L）
- 实时预览表达式字段值
- 反解析：输入 Cron 表达式回填到 UI
- 最近 8 次运行时间预览
- 内置常用表达式示例和字段说明

### 6. 文本处理

常用文本批量处理工具。

- 去除所有空格
- 去换行（合并为单行）
- 按行去重
- Unicode ↔ 中文互转
- 实时字符数统计
- 结果一键复制

## 技术栈

- Electron 33.x — 桌面应用框架
- 原生 HTML/CSS/JS — 无前端框架依赖
- electron-builder — 打包构建（NSIS 安装包）

## 项目结构

```
├── main.js          # Electron 主进程（窗口管理、菜单、剪贴板）
├── index.html       # 页面结构（六个选项卡）
├── renderer.js      # JSON 模块（虚拟滚动、解析、搜索、编辑、Java Bean）
├── tools.js         # 工具模块（时间戳、翻译、SQL、Cron、文本处理）
├── styles.css       # 全局样式
├── package.json     # 项目配置与构建脚本
```

## 快速开始

> 前提：Node.js 已安装（如安装在 `D:\nodejs`，需确保 PATH 包含该路径）

```bash
# 安装依赖
npm install

# 开发运行
npm start

# 构建免安装版
npm run build

# 构建安装包（NSIS）
npm run build:exe
```

构建产物在 `dist/` 目录下，安装包为 `开发工具箱 Setup x.x.x.exe`。
