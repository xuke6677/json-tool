# JSON格式化工具

一款简洁高效的 JSON 在线解析桌面工具，支持格式化、压缩、语法高亮、关键字搜索等功能。

## 功能特性

- ✨ JSON 格式化/压缩
- ✨ 语法高亮显示
- ✨ 实时校验
- ✨ 关键字搜索（支持上下导航）
- ✨ 折叠/展开层级
- ✨ 行号显示
- ✨ 错误提示（显示错误行号）
- ✨ 复制到剪贴板

## 开发

### 环境要求

- Node.js 14+
- npm 6+

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm start
```

### 构建 EXE

```bash
npm run build
```

构建完成后，可执行文件位于：
- 便携版：`dist4/win-unpacked/JSON格式化工具.exe`
- 安装包：`dist4/JSON格式化工具 Setup 1.0.0.exe`

## 使用说明

1. 在左侧输入框粘贴 JSON 字符串
2. 右侧自动显示解析结果
3. 支持关键字搜索：输入关键字后点击搜索按钮或按回车
4. 使用上下箭头导航匹配结果
5. 点击折叠/展开按钮控制 JSON 层级的显示
