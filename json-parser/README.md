# JSON 格式化工具

基于 Electron 的桌面端 JSON 解析工具，支持格式化、压缩、校验、语法高亮、折叠展开、搜索、内联编辑、转义处理、Java Bean 转换等功能。采用虚拟滚动技术，可流畅处理大体量 JSON 数据。

## 功能特性

### 核心功能
- **格式化** — 将压缩的 JSON 字符串格式化为缩进 2 空格的可读格式
- **压缩** — 将格式化的 JSON 压缩为单行
- **校验** — 验证 JSON 语法是否合法，错误时提示行号和列号
- **复制** — 一键复制输入区或输出区内容到剪贴板
- **清空** — 清除输入区内容

### 转义处理
- **转义** — 将 JSON 字符串转义（`JSON.stringify`）
- **去转义** — 还原被转义的 JSON 字符串

### 输出区功能
- **语法高亮** — key、string、number、boolean、null 分色显示
- **折叠/展开** — 支持单个节点折叠展开，支持全部折叠/全部展开
- **内联编辑** — 点击 key 或 value 可直接编辑，修改后自动同步到输入区
- **时间戳识别** — 鼠标悬停在 13 位时间戳上自动显示可读日期时间
- **转 Java Bean** — 将 JSON 结构转换为 Java Bean 类代码，支持嵌套对象和数组

### 搜索功能
- 左侧输入区和右侧输出区各有独立搜索栏
- 支持关键字搜索、上一个/下一个导航、搜索计数显示
- 右侧搜索会自动展开被折叠的匹配节点

### 性能优化
- **虚拟滚动** — 右侧输出区只渲染可视区域的行，DOM 节点数始终可控
- **行号虚拟化** — 行号与内容同步虚拟渲染
- **数组拼接渲染** — HTML 生成使用数组 push + join 替代字符串拼接
- **事件委托** — 折叠、编辑、时间戳等事件统一委托处理
- **防抖输入** — 输入变化 300ms 防抖后再解析渲染

### 界面特性
- 左右双栏布局，可拖动分隔条调整宽度
- 底部状态栏显示字符数、行数、数据大小
- 菜单栏含关于信息和功能使用说明

## 技术栈

- **Electron** 33.x — 桌面应用框架
- **原生 HTML/CSS/JS** — 无前端框架依赖
- **electron-builder** — 打包构建

## 项目结构

```
├── main.js          # Electron 主进程（窗口管理、菜单、剪贴板）
├── index.html       # 页面结构
├── renderer.js      # 渲染进程（虚拟滚动、JSON解析、搜索、编辑、Java Bean转换）
├── styles.css       # 样式
├── package.json     # 项目配置与构建脚本
```

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
npm start
```

### 构建
```bash
# 构建为目录（免安装版）
npm run build

# 构建为安装包（NSIS）
npm run build:exe
```

构建产物在 `dist4_backup/win-unpacked/` 目录下。
